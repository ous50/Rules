# Rules
Clash自用规则组，集成众多大佬的规则


|文件或文件夹名称|描述|
|:--:|:--:|
|MyConvert|是我自己用的订阅转换|
|Rules_Config|通用clash配置|
|Rules_Convert|订阅转换配置(需要按需求自行修改)|
|Rules_Parsers|预处理配置(需要按需求自行修改)|
|Rules_Provider|规则|
|config.yaml|一个另类的配置可以(需要按需求自行修改)|
